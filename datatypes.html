Data Types

Data Types:- Data type indicates the type of information.

The Data type represent a type of value that determines which operation can be performed per data.

▪️Text Type: str

▪️Numeric Types: int, float, complex

▪️Sequence Types: list, tuple, range

▪️Mapping Type: dict

▪️Set Types: set, frozenset

▪️Boolean Type: bool

▪️Binary Types: bytes, bytearray, memoryview
